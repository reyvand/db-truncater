# db-truncater

[ ! ] Truncater DB v1.1<br>
[ ! ] Coded by kod0kk - [github.com/reyvand](https://github.com/reyvand/)

[ ! ] Example<br>
* Check available databases
  * $ ./truncater-db -check
* Truncate all table in database
  * $ ./truncater-db -h host -u user -p pass -d database
* Requirements :
  * mysqli extension


_*works well on php7_
